# My AstroNvim Configuration

**NOTE:** This is for AstroNvim v5+

A customized [AstroNvim](https://github.com/AstroNvim/AstroNvim) configuration for PHP, Laravel, Next.js, TypeScript, Python, and C++ development.

## 🛠️ Installation

#### Clone the repository

```shell
git clone https://github.com/loka1/nvim ~/.config/nvim
```

#### Start Neovim

```shell
nvim
```

## 📦 Supported Languages

- **PHP & Laravel** - Intelephense, blade-formatter
- **JavaScript/TypeScript** - tsserver, ESLint, Prettier (Next.js ready)
- **HTML/CSS** - Tailwind CSS, Emmet, colorizer
- **Python** - basedpyright, black, isort, ruff
- **C/C++** - clangd, cmake, cppdbg

## ⚡ Features

- Format on save
- Git integration with gitsigns
- Auto-close tags for HTML/JSX/Blade
- TODO/FIXME highlight
- TypeScript shortcuts (organize imports, remove unused)
